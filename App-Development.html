4. MainActivity Code
Add Bluetooth functionality and button listeners in MainActivity.java:

<pre><code>

package com.example.rccarcontroller;

import android.bluetooth.BluetoothAdapter;
import android.bluetooth.BluetoothDevice;
import android.bluetooth.BluetoothSocket;
import android.os.Bundle;
import android.util.Log;
import android.view.View;
import android.widget.Button;
import androidx.appcompat.app.AppCompatActivity;
import java.io.IOException;
import java.io.OutputStream;
import java.util.UUID;

public class MainActivity extends AppCompatActivity {

    private static final String TAG = "RC Car Controller";
    private static final String DEVICE_ADDRESS = "XX:XX:XX:XX:XX:XX"; // Replace with your HC-05 address
    private static final UUID MY_UUID = UUID.fromString("00001101-0000-1000-8000-00805F9B34FB");

    private BluetoothAdapter bluetoothAdapter;
    private BluetoothSocket bluetoothSocket;
    private OutputStream outputStream;

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);

        bluetoothAdapter = BluetoothAdapter.getDefaultAdapter();
        connectToDevice();

        Button buttonForward = findViewById(R.id.buttonForward);
        Button buttonBackward = findViewById(R.id.buttonBackward);
        Button buttonLeft = findViewById(R.id.buttonLeft);
        Button buttonRight = findViewById(R.id.buttonRight);
        Button buttonServo = findViewById(R.id.buttonServo);

        buttonForward.setOnClickListener(new View.OnClickListener() {
            @Override
            public void onClick(View v) {
                sendCommand('F');
            }
        });

        buttonBackward.setOnClickListener(new View.OnClickListener() {
            @Override
            public void onClick(View v) {
                sendCommand('B');
            }
        });

        buttonLeft.setOnClickListener(new View.OnClickListener() {
            @Override
            public void onClick(View v) {
                sendCommand('L');
            }
        });

        buttonRight.setOnClickListener(new View.OnClickListener() {
            @Override
            public void onClick(View v) {
                sendCommand('R');
            }
        });

        buttonServo.setOnClickListener(new View.OnClickListener() {
            @Override
            public void onClick(View v) {
                sendCommand('S');
            }
        });
    }

    private void connectToDevice() {
        BluetoothDevice device = bluetoothAdapter.getRemoteDevice(DEVICE_ADDRESS);
        try {
            bluetoothSocket = device.createRfcommSocketToServiceRecord(MY_UUID);
            bluetoothSocket.connect();
            outputStream = bluetoothSocket.getOutputStream();
        } catch (IOException e) {
            Log.e(TAG, "Error connecting to device", e);
        }
    }

    private void sendCommand(char command) {
        try {
            outputStream.write(command);
        } catch (IOException e) {
            Log.e(TAG, "Error sending command", e);
        }
    }

    @Override
    protected void onDestroy() {
        super.onDestroy();
        try {
            bluetoothSocket.close();
        } catch (IOException e) {
            Log.e(TAG, "Error closing socket", e);
        }
    }
}

Summary
This app connects to your RC car's Bluetooth module.
It provides buttons for controlling the car's movement and the servo.
Ensure the DEVICE_ADDRESS in the code matches your HC-05 Bluetooth module's MAC address.







Explanation of the Code
AndroidManifest.xml:

Permissions: Grants the app permissions to use Bluetooth and Bluetooth admin features.
activity_main.xml:

Layout: Defines the user interface with buttons for controlling the RC car (Forward, Backward, Left, Right, Servo).
MainActivity.java:

Bluetooth Setup:

BluetoothAdapter: Initializes Bluetooth adapter to manage Bluetooth settings.
BluetoothDevice: Represents the remote Bluetooth device (HC-05 module).
BluetoothSocket: Manages the connection to the remote Bluetooth device.
OutputStream: Used to send data to the Bluetooth device.
onCreate() Method:

setContentView: Sets the layout for the activity.
connectToDevice(): Establishes the Bluetooth connection to the HC-05 module.
findViewById: Finds the buttons in the layout.
setOnClickListener: Sets up listeners for each button to send the corresponding command when pressed.
connectToDevice() Method:

getRemoteDevice: Retrieves the Bluetooth device using its MAC address.
createRfcommSocketToServiceRecord: Creates a Bluetooth socket for communication.
connect: Establishes the connection to the remote device.
getOutputStream: Gets the output stream to send commands to the device.
sendCommand() Method:

write: Sends the command character to the Bluetooth device.
onDestroy() Method:

close: Closes the Bluetooth socket when the app is destroyed to clean up resources.
Tutorial: How to Create the App in Android Studio
Install Android Studio:

Download and install Android Studio.
Create a New Project:

Open Android Studio and select "Start a new Android Studio project".
Choose "Empty Activity" and click "Next".
Name your application (e.g., "RCCarController") and set the package name (e.g., "com.example.rccarcontroller").
Choose the language (Java) and the minimum API level (e.g., API 21: Android 5.0 Lollipop).
Click "Finish" to create the project.
Modify AndroidManifest.xml:

Add the required Bluetooth permissions inside the  tag.
Design the Layout:

Open activity_main.xml in the res/layout folder.
Design the user interface by adding buttons for controlling the RC car.
Implement Bluetooth Functionality:

Open MainActivity.java in the java folder.
Add the necessary imports for Bluetooth functionality.
Implement the methods and logic as explained earlier.
Test the App:

Connect your Android device via USB and enable developer mode.
Ensure Bluetooth is enabled on your device.
Pair your device with the HC-05 module.
Run the app from Android Studio and install it on your device.
Test the buttons to ensure they send the correct commands to the RC car.
